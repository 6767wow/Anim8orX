# Anim8orX v0.1.0

Initial foundation release.

## Added

- Retained-mode UI architecture blueprint for a UniverseLib-styled editor shell.
- C++17 viewport camera controller.
- Unity-style navigation input model:
  - RMB + WASD fly
  - Alt + left mouse orbit
  - Alt + middle mouse pan
  - scroll dolly
  - F focus hook
- C++17 `.an8` lexer/parser.
- Basic object, mesh, points, and faces parsing.
- Face fan triangulation helper for Vulkan-style vertex/index buffer upload.
- CLI parser sandbox and sample cube `.an8` asset.

## Notes

This release intentionally does not include the Vulkan renderer yet. The included code establishes the import and camera foundations that the Vulkan viewport will consume next.
