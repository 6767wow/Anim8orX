# Anim8orX v0.6.3

Imported object display and viewport lag cleanup.

## Fixed

- Imported meshes now render as filled flat-shaded objects by default instead of only looking like wire/bone outlines.
- Empty geometry nodes are skipped in the hierarchy so non-renderable helper chunks are less likely to appear as the main object.
- Mesh display names now prefer real object/mesh names and avoid generic parser fallback names when better names exist.
- Large selected meshes no longer redraw every selected edge by default; they use a cheap projected selection rectangle unless wireframe is enabled.

## Added

- `Options > Wireframe` toggle for explicit edge/wire viewing.
- `Options > Flat Shaded` toggle for filled face rendering.

## Verification

## Notes

This is still the native GDI editor shell while the long-term Vulkan viewport is pending, but imported models should now appear as actual filled objects and interact with much less redraw cost.
