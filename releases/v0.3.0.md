# Anim8orX v0.3.0

Anim8or-style shell expansion.

## Added

- Reworked the default UI to be viewport-first like Anim8or.
- Added native menu categories:
  - File
  - Edit
  - Mode
  - Object
  - Options
  - View
  - Build
  - Scripts
  - Render
  - Window
  - About
- Added compact dark command strip with icon-style buttons and frame markers.
- Added right-aligned Object, Figure, Sequence, and Scene mode tabs with orange active state.
- Added a slim left modeling rail with grouped tools:
  - Mode
  - Coord
  - Axis
  - Tools
  - UV
  - Shapes
  - Rig
  - Keys
- Added bottom status/object strip similar to Anim8or.
- Added UniverseLib-style dock entry points for Explorer, Inspector, Console, Materials, and Timeline without covering the default viewport.
- Added a gray grid viewport with orange Front label, axis marker, selected wireframe cube, and Unity camera control hint.
- Added clickable top command icons and rail tool buttons.
- Added menu mode commands for Object/Figure/Sequence/Scene.

## Still pending

- Real Vulkan renderer swap-in.
- Actual geometry command implementations for extrude, lathe, subdivision, bevel, primitive creation, rigging, sequencing, materials, and scene rendering.
- Full retained widget object model extraction from the current native shell.

## Verification

- Built with MSVC/CMake Release configuration.
- Ran `Anim8orX.exe --smoke-test`.
- Extracted release zip and verified the GUI window remains open.
