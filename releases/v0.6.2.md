# Anim8orX v0.6.2

Viewport interaction lag reduction.

## Optimized

- Camera/navigation frames now invalidate only the viewport rectangle.
- Paint handling now detects viewport-only dirty regions and redraws only the viewport into the persistent back buffer.
- Partial repaint blits only the invalidated rectangle instead of the full window.
- The hierarchy, inspector, top bar, console, and status strip are no longer redrawn every camera frame.
- Capture/release, hover border changes, timer-driven camera movement, and Escape cancellation now use viewport-only invalidation where possible.
