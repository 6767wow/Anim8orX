# Anim8orX v0.6.1

Performance optimization pass.

## Optimized

- Removed the always-on 60 Hz viewport timer while the editor is idle.
- Added a dynamic frame timer that only wakes during viewport navigation, wheel zoom/speed changes, or focus actions.
- Added cached GDI pens and brushes instead of allocating and deleting handles for every line, rectangle, and toolbar glyph.
- Added a persistent back buffer so the full-window compatible bitmap is not recreated every repaint.
- Cached camera projection basis once per paint instead of recomputing forward/right/up/FOV math for every projected point.
- Changed mesh wire drawing to project each mesh vertex once per frame and reuse those projected points across all face edges.
- Kept the timer alive while RMB viewport navigation is active so WASD fly remains responsive.

## Notes

This is still the retained GDI editor shell. The later Vulkan viewport brick should replace the wire renderer, but this pass removes the worst shell-side lag so the current build is much more usable.
