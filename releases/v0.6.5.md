# Anim8orX v0.6.5

Still-frame model display correction.

## Fixed

- The viewport no longer drops faces while the camera is still. Large `.an8` models now draw complete filled faces at rest instead of looking chopped up or skeletal.
- The low-detail face budget is now only used during active navigation, where responsiveness matters most.
- Selected meshes no longer draw an automatic edge overlay by default. Use `Options > Wireframe` when explicit wire display is wanted.
- Diffuse `texturename { ... }` entries inside material `diffuse` blocks are now parsed.
- Bump/normal maps no longer overwrite an already parsed diffuse texture reference for the same material.
