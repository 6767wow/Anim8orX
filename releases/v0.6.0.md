# Anim8orX v0.6.0

Functional editor brick.

## Added

- Added a separate selectable Hierarchy panel beside the Anim8or-style tool rail.
- Added working File commands:
  - `Open .an8...`
  - `Import`
  - `Save`
  - `Save As...`
  - `Export`
- Added a simple `.an8` mesh writer for saving the current object/mesh document.
- Added working Build commands:
  - cube
  - sphere
  - cylinder
  - cone
  - torus
  - text helper
  - bone helper
  - camera helper
  - light helper
- Added working Object commands:
  - delete
  - extrude
  - inset
  - subdivide faces
  - lathe
  - mirror
- Added viewport option toggles for grid, grid snapping, axes, normals, and backface culling.
- Added `Frame Selection` view command.
- Added Delete, Ctrl+N, Ctrl+O, and Ctrl+S keyboard paths.
- Added RMB+mouse wheel fly-speed adjustment for Unity-style fly navigation.
- Added parser sample files for nested Anim8or face point-data and primitive component chunks.

## Parser

- Parses `group` chunks inside objects.
- Parses `name` chunks on objects, groups, and meshes.
- Parses common `base { origin { ... } }` translation data for mesh and primitive components.
- Parses Anim8or nested face point-data records such as `( (2 6) (0 5) (1 7) )` by taking the first value in each point-data tuple as the point index.
- Converts common saved primitive components into renderable meshes:
  - `cube`
  - `sphere`
  - `cylinder`

## Fixed

- Hierarchy UI now actually renders and can select mesh nodes.
- Many menu items that previously only logged a pending message now mutate the document or viewport state.
- `.an8` files using standard nested face point-data no longer import as blank meshes.
- `.an8` files saved with primitive component chunks are less likely to open as empty documents.
- `File > Save` no longer remains a dead menu entry.
- Scene helpers can now be added and selected from the Scene editor workflow.

## Verification

- Built with MSVC/CMake Release configuration.
- Ran bundled cube smoke test:
  - `Anim8orX.exe --smoke-test .\examples\cube.an8`
- Ran nested face-data smoke test:
  - `Anim8orX.exe --smoke-test .\examples\nested-face.an8`
- Ran primitive component smoke test:
  - `Anim8orX.exe --smoke-test .\examples\primitive-components.an8`

## Still Not Complete

This is not yet a finished full Anim8or replacement. Vulkan rendering, real point/edge/face picking, Catmull-Clark subdivision, full rig skinning, sequence playback, material texture pipelines, and production rendering are still future bricks.
