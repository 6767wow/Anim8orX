# Anim8orX v0.2.0

First native editor-shell release.

## Added

- Replaced the CLI-only launch with a native Windows GUI application.
- Added a custom dark retained-style editor layout inspired by UniverseLib:
  - persistent top navigation
  - Object, Figure, Sequence, and Scene mode tabs
  - workspace tabs for Viewport, Hierarchy, Inspector, and Console
  - bordered hierarchy, inspector, console, toolbar, and viewport regions
- Added an Anim8or-style default cube scene on launch.
- Added wireframe viewport drawing for imported `.an8` meshes.
- Added scene hierarchy listing for loaded object/mesh nodes.
- Added inspector data for transform, mesh counts, and camera yaw/pitch.
- Kept Unity-style viewport controls:
  - right mouse + WASD/QE fly
  - Alt + left mouse orbit
  - Alt + middle mouse pan
  - mouse wheel dolly
  - F focus on selection
- Added `--smoke-test` startup verifier for automated release checks.

## Notes

This release is the first visible editor shell. Vulkan rendering and full modeling tools are still the next implementation layer, but the app now launches like an editor instead of a parser test.
