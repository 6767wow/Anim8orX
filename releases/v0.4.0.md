# Anim8orX v0.4.0

Anim8orX property deck expansion.

## Added

- Added an Anim8orX-branded right-side property deck.
- Added property page tabs:
  - Setup
  - View
  - Material
  - Object
  - Figure
  - Sequence
  - Scene
  - Render
- Added visible GUI controls for the provided checklist:
  - grid width, grid snap, snap size, angle snap, angle size, backface culling, autosave interval
  - wireframe, flat shaded, smooth shaded, textured, grid, axes, normals, normal scale
  - material name, ambient/diffuse/specular/emissive colors, brilliance, roughness, transparency, texture maps
  - cube/sphere/cylinder/cone primitive dimensions and divisions
  - lathe degrees, lathe segments, axis selection, subdivision level, smooth edge angle
  - bone length, influence radii, per-axis rotational DOF enable/min/max
  - sequence name, frame count, FPS, interpolation, TCB sliders
  - scene instance transform, camera focal/FOV/clipping, light type/color/intensity/cones/shadows
  - render pipeline selector, output size, aspect lock, frame range, AA settings, ART quality options
- Mode changes now switch the property deck to the matching editor page.
- Property page tabs are clickable.
- Restored a bottom dock while preserving the viewport-first modeling workspace.

## Notes

This release adds the full GUI surface for the requested Anim8or-compatible properties. Many controls are currently retained UI stubs and still need to be wired into persistent document state, undoable commands, Vulkan rendering, and real modeling/render operations.
