# Anim8orX v0.6.7

Viewport click hotfix.

## Fixed

- Plain left-clicks in the viewport no longer start mouse capture or navigation-preview mode.
- Plain middle-clicks in the viewport no longer start mouse capture unless Alt is held.
- Unity-style navigation now matches the advertised controls more closely:
  - `Alt+LMB` orbits.
  - `Alt+MMB` pans.
  - `RMB` captures for fly/look navigation.
- Lost mouse capture, window focus loss, or cancel-mode messages now clear held mouse state so the viewport cannot get stuck in a reduced preview/navigation state.
