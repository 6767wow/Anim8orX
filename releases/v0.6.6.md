# Anim8orX v0.6.6

Texture/depth preview correction for imported `.an8` models.

## Added

- The viewport now keeps imported UV coordinates in the renderable mesh cache instead of dropping them after parsing.
- Resolved diffuse textures are downsampled into lightweight CPU preview buffers during file load.
- Imported textured faces now cache a UV-sampled diffuse color instead of using only one whole-texture average.
- Filled faces are collected and painter-sorted back-to-front before drawing, which improves overlap ordering for complex models in the current GDI preview shell.

## Fixed

- Large imported models should look less flat and less incorrectly layered when materials/textures are available.
- Bump/normal maps remain ignored for diffuse preview color, so they no longer tint the visible mesh surface.
- Navigation still uses the reduced face budget while mouse/camera movement is active, keeping viewport movement responsive.

## Notes

This is still a preview-renderer brick, not the final Vulkan renderer. It does not yet do per-pixel UV rasterization, true z-buffering, or skinned figure pose playback.
