# Anim8orX v0.6.8

Solid viewport display hotfix.

## Fixed

- `Options > Flat Shaded` no longer makes imported models collapse into a line-heavy/wireframe-looking view.
- Filled faces now render unconditionally in the viewport. The flat-shading flag only controls simple per-face lighting.
- Wireframe is now treated as an overlay mode, not as a replacement for the solid mesh preview.
- The right-side display option label now says `Flat Lighting` to match what the current renderer can safely do.

## Verification

- Built with MSVC/CMake Release configuration.
- Opened the Baldi `.an8` file and intentionally toggled the former flat-shaded command off.
- Captured the viewport after the toggle and confirmed the character stayed as a solid filled model instead of turning into the broken edge-only view.
- Ran parser smoke tests on bundled examples and real `.an8` files.
